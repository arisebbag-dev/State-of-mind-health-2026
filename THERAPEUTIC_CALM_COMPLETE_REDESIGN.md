# 🎨 COMPLETE REDESIGN: Therapeutic Calm Color Palette

## What's Changed

### **Color Palette: Therapeutic Calm**
This palette is specifically designed to be soothing and calming for psychiatric practice visitors.

**Primary Color:** Soft Slate Blue (#5B7C99)
- Use for: Headings, primary buttons, important elements
- Conveys: Trust, calm, professionalism, clarity

**Secondary Color:** Sage Green (#7BA384)
- Use for: Accents, hover states, featured elements
- Conveys: Nature, healing, growth, wellness

**Accent Color:** Warm Cream (#F5F3EF)
- Use for: Background, safe space feeling
- Conveys: Warmth, comfort, cleanliness

**Text Colors:**
- Deep text: Ink (#2D3E4A) - for maximum readability
- Soft text: Ink Soft (#5A6B77) - for secondary content
- Muted text: Ink Muted (#8A9AA5) - for tertiary content

---

### **Font Updates**

**Display Font:** Crimson Text (Serif)
- Elegant, warm, sophisticated
- Perfect for headings and taglines
- Creates professional yet approachable feel

**Body Font:** Inter (Sans-serif)
- Highly readable, modern, clean
- Comfortable for extended reading
- Light weight for softer appearance
